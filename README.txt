Данный юзер-бот написан для удобной рассылки сообщений пользователям
github: https://github.com/coldiceiceice



1. Переходим на сайт https://my.telegram.org/auth и авторизируемся
2. Переходим в API development tools и копируем api_id и api_hash
2.1 Если api_id и api_hash отсутсвует, а открылось что-то не понятное, то, во-первых,
    не паникуем и просто регистрируемся, вписываем нужные там данные, и после этих манипуляций
    появятся заветные hash и id.
3. Открываем config с помощью редактора
4. Вставляем api_id и api_hash с сайта
5. Сохраняем, запускаем run.bat
6. В телеге открыем "избранное" (мы же не хотим спамить командами другим людям)
7. Пишем .help, наблюдаем доступные команды.
Всё

Для заполнения списка через блокнот впишите все имена в people.txt по одному имеми в строчку.
